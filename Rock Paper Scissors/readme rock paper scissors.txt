This Python script simulates a game of Rock, Paper, Scissors between a user and a computer.

How it works:

User Input: The script prompts the user to choose Rock, Paper, or Scissors.
Computer Choice: The script randomly generates a choice for the computer.
Game Logic: The script determines the winner based on the classic Rock, Paper, Scissors rules.
Output: The script displays the choices made by the user and the computer, as well as the outcome of the game.
